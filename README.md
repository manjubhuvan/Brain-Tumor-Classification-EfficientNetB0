# Brain Tumor Classification 

This project focuses on classifying brain tumor MRI images using a deep learning model built with **EfficientNetB0** and **Transfer Learning**.  
The model predicts four classes: **Glioma**, **Meningioma**, **Pituitary Tumor**, and **No Tumor**.

## Project Overview
- MRI-based brain tumor classification  
- Uses a pretrained EfficientNetB0 model  
- Includes Grad-CAM visualizations for explainability  
- High accuracy achieved with fine-tuning and augmentation
